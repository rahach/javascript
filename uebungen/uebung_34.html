Neu in HTML5 sind folgende semantischen Container-Elemente

header, nav, main, section, article, aside, footer

Das ist ein hervorgehobener Text mit <em> - Der physische Auszeichner dazu ist: <i> wie "italic"

Absätze werden mit <p> definiert. Diese sind Blockelemente, die keine weiteren Blockelemente enthalten dürfen. Sie besitzen im Browser-Style Außen-Abstände nach oben und unten.

Ein <div> - Division - ist ein Blockelement ohne Formatierungen. Kann alle weiteren HTML-Elemente enthalten.

Ein Attribut "target" kann folgende Inhalte haben:

  • _blank - Open in a new browser window
  • _parent - Open in the parent frame or window
  • _self - Open in the current window or frame (default)
  • _top - Open in the topmost frame, thus replacing the contents of the window
  • iframe-Attribut "name" - Open in the <iframe> element with matching name attribute

Images werden mit <img> definiert. Sie besitzen zwei Pflichtattribute: src und alt. Mögliche Formate sind: gif, jpg, png, svg. Images sind Inline-Elemente

Formularelemente

Zum Senden der Formulardaten mit "submit" werden im form-tag folgende Attribute verwendet: action, method, enctype (nur bei File Upload)

form, input, label, select, option, textarea, button, fieldset, legend

Attribute type-values (alt): text, checkbox, radio, submit, reset, button, password, image, hidden, file (zusätzlich Attr. enctype Pflicht)

Attribut type-values (neu in HTML5): email, url, color, range, search, date, datetime, time, number, tel

datalist (Alternative zu select)


Imagemap

Eine map-Definition beinhaltet ein oder mehrere <area>-Tags. Diese besitzen die folgenden Pflichtattribute: shape, coords, href, alt. Im Attribut Shape sind folgende Attribute möglich: rect circle, poly. Das Image wird mit einer Map über das Attribut "name" der Map und das Attribut "usemap" (# nicht vergessen) im <img> Tag verknüpft.

Multimedia

Video-Dateien werden mit dem HTML5-Element <video> und <source> eingebunden

Video Formate, die Browser unterstützen sind mp4, webm, ogv

Attribute des Video-Tags: width, height, src, autocomplete, controls, poster, loop, preload, muted

Audio-Dateien werden mit dem HTML5-Element <audio> und <source> eingebunden

Audio Dateien, die Browser unterstützen sind mp3, mp4, ogg, wav

Andere Multimedia-Dateien (z.B. Flash) werden mit dem HTML5-Element <object> und/oder <embed> eingebunden

Inline-Frames <iframe>

Ein Iframe besitzt die Pflichattribute src, width und/oder height. Optional sind: seamless, sandbox
